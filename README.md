## Installation
**Clone the repository from:**
```
https://github.com/WesselSmit/frontend-applications.git
```

**NPM Package.json**
```
npm install
```

**Usage**

Navigate to the repository

```
npm run serve
```

You can now find the application with the following URL:
```
localhost:8080
```
